Программа установки обнаружила, что некоторые сетевые компоненты несовместимы с Microsoft Windows XP. При наличии диска, содержащего файлы, необходимые для обновления данного компонента, нажмите кнопку "Установить с диска" и найдите эти файлы. При отсутствии необходимых файлов компонент, возможно, не будет правильно обновлен или некоторые параметры настройки могут будут утеряны.

Иногда программа установки не может определить, правильно ли будет обновлено отдельное сетевое устройство. Чтобы прекратить обновление немедленно, нажмите кнопку "Отмена" для выхода из программы установки.

Полный перечень устройств, поддерживаемых Windows 2000, содержится в списке совместимого оборудования Microsoft Windows 2000 Hardware Compatibility List на веб-узле http://www.microsoft.com/hwtest/hcl.
