Этот компонент использует драйвер устойства, называемый драйвером
фильтра файловой системы. Драйверы фильтра файловой системы предоставляют возможность другим компаниям писать программы, которые обеспечивают доступ к файлам на жестких дисках. Обычно они
применяются в таких программных продуктах, как антивирусные программы и программы архивации, но также могут быть 
использованы в устройствах хранения информации, например устройствах чтения компакт-дисков, устройствах смены дисков и ленточных устройствах. 

Этот драйвер не был проверен в Microsoft Windows 2000, поэтому не следует продолжать обновление, если данный драйвер загружен. Компьютер не содержит достаточных сведений для определения программного продукта использующего этот драйвер.

Выполните одно из следующих действий:

1. Определите и удалите программу, использующую этот драйвер. Иногда драйвер позволяет определить программный продукт его использующий. Полный список драйверов и программ их использующих можно найти в «Замечаниях к данному выпуску» на компакт-диске Windows 2000 или на веб-узле http://www.microsoft.com/windows/server/deploy/fileprint/fsfilteruse.

2. В окне «Панель управления» дважды щелкните значок «Устройства». Найдите запись для этого драйвера и нажмите кнопку <B>Автозагрузка<B>, а затем «Выключен». Нажмите кнопки OK, «Close» и перезагрузите компьютер. Перезапустите программу установки Windows 2000. Можно продолжить обновление, но программный продукт,
использующий этот драйвер будет работать неправильно.

Полный список устройств, поддерживаемых Windows 2000, содержится в списке совместимого оборудования Microsoft Windows 2000 Hardware Compatibility List на веб-узле http://www.microsoft.com/hwtest/hcl. Компакт-диск Windows 2000 содержит копию этого списка (диск:\Support\hcl.txt) на момент выхода Windows 2000.
