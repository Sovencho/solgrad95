При использовании установленного драйвера в Microsoft Windows ХР перечисленные ниже принтеры Xerox будут работать неправильно.

  • Xerox DocuPrint P12

Замените драйвер обновленной версией, в противном случае программа установки Windows XP установит совместимый драйвер.

Для получения последней версии драйверов принтера посетите веб-узел Xerox. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.
