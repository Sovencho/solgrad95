Сведения о совместимости

Установленный драйвер расширенной клавиатуры Hewlett-Packard Extended Keyboard несовместим с Microsoft Windows ХР и будет отключен при обновлении.

Windows ХР установит стандартный драйвер клавиатуры, при этом не будет возможности использовать функции дополнительных клавиш данной модели клавиатуры.

Для получения дополнительных сведений об этом драйвере посетите веб-узел разработчика http://go.microsoft.com/fwlink/?linkid=380. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
