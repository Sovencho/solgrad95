Установленный драйвер устройства Low Speed IRDA несовместим с данной версией Microsoft Windows и будет отключен при обновлении. Для полного устранения конфликта данное устройство следует удалить.

Чтобы удалить это устройство, выполните следующие действия

  1. Вставьте в дисковод А дискету IBM IRDA Driver.
  2. Нажмите кнопку Пуск и выберите команду Выполнить.
  3. В поле Открыть, наберите A:\INSTALL.EXE и нажмите кнопку OK.
  4. Нажмите кнопку DeInstall.
  5. Нажмите кнопку Добавить/удалить.
  6. Перезагрузите компьютер и запустите программу установки Windows ХР.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
