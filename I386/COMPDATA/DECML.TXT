Сведения о совместимости

Компания-изготовитель компьютера Digital Equipment Corporation, отделение Compaq, попросила корпорацию Майкрософт прекратить поддержку данной модели компьютера. Корпорация Майкрософт не предоставляет файлы, необходимые для запуска Microsoft Windows ХР на данном компьютере.

Для получения дополнительных сведений обратитесь к изготовителю данного компьютера.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
