Сведения о совместимости

Установленный драйвер для программы Siemens PCI Driver, входящей в пакет DeskView компании Siemens, несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
