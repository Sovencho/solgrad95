Сведения о совместимости

Установленное программное обеспечение ASPI для сканера Microtek C3 несовместимо с данной версией Microsoft Windows и будет отключено при обновлении.

Для получения дополнительных сведений об этом драйвере посетите веб-узел Microtek. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
