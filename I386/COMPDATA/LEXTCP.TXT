Сведения о совместимости

Монитор порта Lexmark TCP/IP Network Port больше недоступен в данной версии Microsoft Windows; программное обеспечение, связанное с с данным портом, после обновления запускаться не будет. При использовании данного монитора порта определите, поддерживает ли устройство стандартный монитор порта TCP/IP Microsoft, и измените параметры конфигурации для каждой очереди печати после завершения установки.

Если поддерживаемые данным устройством протоколы неизвестны, для получения дополнительных сведений посетите веб-узел Lexmark. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
