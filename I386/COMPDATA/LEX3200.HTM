При использовании установленного драйвера в Microsoft Windows ХР перечисленные ниже принтеры Lexmark будут работать некорректно.

  • Lexmark 3200 Series ColorFine

Замените драйвер обновленной версией, в противном случае программа установки Windows XP установит совместимый драйвер.

Для получения последней версии драйверов принтера посетите веб-узел Lexmark. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.
