Сведения о совместимости

Программа Easy Access Buttons несовместима с Microsoft Windows ХР и будет отключена при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Для получения дополнительных сведений о данном устройстве или версии программы, работающей в среде Windows XP, посетите веб-узел разработчика http://go.microsoft.com/fwlink/?linkid=370. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный список программ, поддерживаемых Windows ХР, содержится в списке совместимых программных продуктов Microsoft Windows XP Product Compatibility List на веб-узле корпорации Майкрософт.
