Сведения о совместимости

Данное приложение заменило программное обеспечение для входа в систему Windows, (например, приложение для входа в систему с использованием биометрических параметров или смарт-карты). После обновления эта программа может перестать работать, что может сделать невозможным входу в систему. После обновления некоторые функции операционной системы могут оказаться недоступными, даже если данная программа будет работать. Рекомендуется удалить данное программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
