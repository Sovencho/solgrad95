Сведения о совместимости

Установленный драйвер управления электропитанием несовместим с данной версией Microsoft Windows и должен быть удален перед продолжением установки Windows XP.

Для получения дополнительных сведений обратитесь к разработчику этого драйвера.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
