Сведения о совместимости

Установленный драйвер RAID-контроллера Adaptec серии AAA или ARO несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Если компьютер загружается с диска, подключенного к RAID-контроллеру Adaptec серии AAA или ARO, обновление будет невозможно без драйвера, совместимого с данной версией Windows.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
