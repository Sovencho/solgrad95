Сведения о совместимости

Установленный протокол AppleTalk Protocol не полностью совместим с данной версией Microsoft Windows и его необходимо удалить для выполнения обновления. Данный протокол исключен из новой версии Windows.

Для получения дополнительных сведений о данном протоколе посетите веб-узел корпорации Майкрософт. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень протоколов, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
