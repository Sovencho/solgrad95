Программное обеспечение Microsoft Exchange Server 5.5 Standard Edition и Microsoft Exchange Server 5.5 Enterprise Edition несовместимо с данной версией Microsoft Windows. Для использования данного программного обеспечения в среде Windows XP необходимо установить пакет обновления Exchange Server 5.5 Service Pack версии 3 или более поздней.

Для получения сведений о данной программе и загрузки пакета обновления Exchange Server 5.5 Service Pack версии 3 или более поздней посетите веб-страницу Microsoft Exchange Server на веб-узле корпорации Майкрософт. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
