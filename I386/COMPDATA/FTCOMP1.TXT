Сведения о совместимости

Программой установки Microsoft Windows XP обнаружены базовые тома Windows NT 4.0. Эти тома могут содержать простые, составные, зеркальные, чередующиеся тома или тома RAID-5.

Перед установкой Microsoft Windows XP эти тома необходимо преобразовать в динамические. Для преобразования базовых дисков, содержащих тома Windows NT 4.0, в динамические используется средство «Администратор дисков».
