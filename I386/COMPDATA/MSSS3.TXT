Сведения о совместимости

В данной версии Windows не поддерживается следующее программное обеспечение.

 Exchange 5.5
 Exchange 2000 (SP1 и более ранний)
 Site Server 3.0 (SP4 и более ранний)
 System Management Server 2.0 (SP2 и более ранний)
 Proxy Server

Для получения дополнительных сведений о данном программном обеспечении см. страницу Microsoft BackOffice Server www.microsoft.com/backofficeserver. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств и приложений, поддерживаемых данной версией Windows, содержится в списке совместимого аппаратного и программного обеспечения на веб-узле корпорации Майкрософт.
