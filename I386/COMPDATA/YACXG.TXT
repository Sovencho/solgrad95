Сведения о совместимости

В процессе обновления установленные драйверы для программного обеспечения Yamaha AC-XG NT4 Audio Drivers будут заменены обновленными версиями, совместимыми с данной версией Microsoft Windows. При обновлении предыдущая версия будет удалена. Рекомендуется удалить это программное обеспечение с помощью значка «Установка и удаление программ» на панели управления.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
