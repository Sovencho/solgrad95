Установленный на компьютере драйвер дисковода DVD-RAM несовместим с данной версией Microsoft Windows и должен быть отключен перед обновлением. Следует удалить этот драйвер для полного устранения конфликта.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке совместимого оборудования на веб-узле корпорации Майкрософт.
