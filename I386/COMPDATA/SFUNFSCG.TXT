Сведения о совместимости

Установленный драйвер служб Windows для клиента UNIX для NFS несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Для получения дополнительных сведений о данном драйвере посетите веб-узел изготовителя по адресу http://www.microsoft.com/windows2000/sfu. Веб-адреса могут изменяться, поэтому есть некоторая вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
