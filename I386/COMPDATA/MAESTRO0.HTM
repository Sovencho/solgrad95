Установленный драйвер расширенной клавиатуры BTC Extended Keyboard несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя по адресу http://go.microsoft.com/fwlink/?LinkId=525. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
