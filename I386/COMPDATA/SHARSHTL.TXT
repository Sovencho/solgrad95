Сведения о совместимости

Установленный драйвер подключаемого к параллельному порту устройства хранения данных несовместим с данной версией Microsoft Windows и должен быть удален перед продолжением установки.

Если после завершения установки устройство не работает, посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
