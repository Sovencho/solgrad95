При обновлении до операционной системы Windows ХР иногда требуется установка новых драйверов. Для программы Tivoli Remote Control требуются специально разработанные драйверы мыши и клавиатуры, которые не были обнаружены на данном компьютере. Для работы с программой Tivoli Remote Control в операционной системе Windows ХР требуется замена текущих драйверов мыши и клавиатуры обновленными.

Новые драйверы Tivoli Remote Control, поддерживаемые операционной системой Windows ХР, доступны в виде программы обновления Patch 3.6.5-RCL-0003, которую можно загрузить с веб-узла изготовителя. Для загрузки этих драйверов необходимо иметь допустимый идентификатор пользователя службы поддержки компании Tivoli.

Если продолжить установку, предварительно не установив новые драйверы, в процессе установки Windows ХР имеющиеся драйверы Tivoli Remote Control будут отключены и работа с Tivoli Remote Control для управления компьютером будет невозможна. Однако, установка программы обновления Tivoli после завершения обновления до Windows ХР восстановит работоспособность устройств Tivoli Remote Control.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
