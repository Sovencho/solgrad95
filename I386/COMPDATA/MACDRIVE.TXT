Сведения о совместимости

Установленный драйвер устройства MacDrive 98 несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Для получения дополнительных сведений о данных программах или версий, работающих в среде Microsoft Windows XP, посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
