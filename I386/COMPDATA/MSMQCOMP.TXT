Сведения о совместимости

На компьютере обнаружена программа Message Queuing 1.0 Information Store. При продолжении обновления до Microsoft Windows XP служба Message Queuing и данные будут недоступны.

Чтобы включить очередь сообщений в среде Windows ХР, после завершения установки Windows ХР запустите служебную программу переноса. Служебная программа преноса может быть запущена с помощью окна настройки сервера, отображаемого при первом запуске Windows ХР.
