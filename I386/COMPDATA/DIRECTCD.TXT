Сведения о совместимости

Программа Adaptec DirectCD несовместима с данной версией Microsoft Windows и будет отключена при обновлении. Рекомендуется удалить данное программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Если данное программное обеспечение поставлялось с компьютером или устройством записи на компакт-диски, обратитесь к изготовителю оборудования за обновлением данного программного обеспечения.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
