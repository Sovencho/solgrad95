Если программа Equis MetaStock 6.5 установлена в Microsoft Windows NT 4.0 (с пакетом обновления версии 3 или более ранней) и создан пароль для программы Downloader for Windows, то при обновлении до Microsoft Windows ХР происходит конфликт. Следует удалить пароль программы Downloader перед продолжением обновления.

Чтобы удалить пароль программы Downloader for Windows

  1. Откройте программу Downloader for Windows.
  2. В меню Tool, выберите команду Vendor Options.
  3. Удалите имя пользователя и пароль, а затем нажмите кнопку OK.

Новый пароль может быть введен после завершения обновления.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
