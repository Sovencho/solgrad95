Программа Siemens MultiPath несовместима с данной версией Microsoft Windows и будет отключена при обновлении. Драйверы Windows ХР не поддерживают несколько контроллеров для одного жесткого диска.

Перед обновлением до Windows XP необходимо удалить данное программное обеспечение и перенастроить систему.

  1. Нажмите кнопку Пуск и в меню Настройка выберите команду Панель управления.
  2. Дважды щелкните значок Установка и удаление программ, а затем удалите программу MultiPath.
  3. После появления соответствующего запроса перезагрузите компьютер.
  4. Перенастройте систему на работу с одним контроллером, подключенным к каждому жесткому диску.
  5. Перезапустите программу установки Windows ХР.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
