Сведения о совместимости

Программа HotSync Manager, выполняющая синхронизацию вычислительного устройства 3Com Palm с компьютером, позволяет синхронизировать устройства Palm с сетью. После обновления до Microsoft Windows ХР сетевая синхронизация будет отключена из-за трудностей в обнаружении устройством основного компьютера.

Чтобы разрешить этот конфликт, синхронизируйте устройство через последовательный порт перед выполнением сетевой синхронизации.

Для получения дополнительных сведений посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
