Программа Compaq Survey Tool несовместима с данной версией Microsoft Windows и будет отключена при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Чтобы удалить это программное обеспечение, выполните следующие действия.

  1. Закройте программу установки Windows ХР.
  2. Нажмите кнопку Пуск и в меню Настройка выберите команду Панель управления.
  3. Дважды щелкните значок Установка и удаление программ.
  4. Выберите Compaq Survey Toolи нажмите на кнопку Добавить/Удалить.
  5. Перезагрузите компьютер и запустите программу установки Windows ХР.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
