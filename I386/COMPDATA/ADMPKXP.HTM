На компьютере обнаружен набор средств администрирования. Перед обновлением удалите средства администрирования с помощью компонента «Установка и удаление программ» на панели управления. Для удаленного администрирования служб сервера с компьютера, работающего под управлением Windows XP Professional, воспользуйтесь средством «Дистанционное управление рабочим столом».

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
