Программа установки Microsoft Windows XP обнаружила, что на данном компьютере выполняется сервер терминалов (формально — «Службы терминалов в режиме сервера приложений»). Сервер терминалов в среде Windows XP Server не поддерживается. Перед продолжением установки необходимо выбрать один их следующих вариантов.

  • Продолжение использования данного компьютера в качестве сервера терминалов.
    1. Отмените обновление.
    2. Установите Windows XP Advanced Server. Кроме того, сервер терминалов входит в состав Windows XP Datacenter Server.
  • Использование данного компьютера для удаленного управления (ранее — «Службы терминалов в режиме удаленного управления»), позволяющего производить до двух удаленный соединений для удаленного управления сервером.
    1. Отмените обновление.
    2. Откройте панель управления, дважды щелкните значок Установка и удаление программ, выберите Добавление и удаление компонентов Windows и нажмите кнопку Далее.
    3. В диалоговом окне Компоненты Windows выберите Режим удаленного управления.
    4. После окончания работы мастера перезагрузите компьютер.
    5. Выполните обновление до Windows XP Server.
