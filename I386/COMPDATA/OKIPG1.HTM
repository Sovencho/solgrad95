При использовании установленного драйвера в Microsoft Windows ХР перечисленные ниже принтеры Oki будут работать неправильно.

  • OKIPAGE 20n PCLXL : Advanced

Замените драйвер обновленной версией, в противном случае программа установки Windows XP установит совместимый драйвер.

Для получения последней версии драйвера посетите веб-узел Oki. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.
