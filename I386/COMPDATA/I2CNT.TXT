Сведения о совместимости

Установленный драйвер устройства VIS SMBus несовместим с Microsoft Windows ХР и будет отключен при обновлении.

Для получения дополнительных сведений о данном драйвере обратитесь к изготовителю устройства или посетите его веб-узел.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
