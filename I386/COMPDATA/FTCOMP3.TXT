Сведения о совместимости

Программой установки Microsoft Windows XP обнаружены базовые тома Windows NT 4.0. Эти тома могут содержать наборы томов, зеркальные наборы томов, наборы томов с чередованием или чередующиеся наборы томов с проверкой четности.

Если установка будет продолжена данные тома будут недоступны в среде Microsoft Windows XP. Утилита ftonline служит для сохранения данных или преобразования этих базовых томов в динамические.

Кроме того, имеется возможность прервать установку, выполнить архивацию содержимого томов и удалить их до повторного запуска программы установки. После завершения установки имеется возможность перенастроить эти тома с использованием динамических дисков и восстановить их содержимое. Для получения дополнительных сведений см. справку Microsoft Windows XP.

Перед запуском программы установки Microsoft Windows XP рекомендуется выполнить архивацию каждого тома.
