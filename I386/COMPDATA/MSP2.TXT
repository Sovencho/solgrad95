Сведения о совместимости

Запущенная версия программы Microsoft Proxy Server 2.0 несовместима с данной версией Microsoft Windows.

Обновление этой программы доступно на веб-узле Microsoft Proxy.

Для получения дополнительных сведений о данной программе или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел разработчика www.microsoft.com/proxy.
