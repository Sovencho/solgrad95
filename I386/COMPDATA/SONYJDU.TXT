Сведения о совместимости

Установленное программное обеспечение Sony Jog Dial Utility несовместимо с данной версией Windows. Перед продолжение установки Windows его необходимо удалить. Удаление может быть выполнено с помощью значка «Установка и удаление программ» на панели управления.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя http://vcl.vaio.sony.co.jp/products/win/index.html. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
