Возможно, установлена не самая последняя версия драйвера цифровой камеры Kodak 4800 или DC-290.  Для обновления драйвера свяжитесь с изготовителем.

Для получения дополнительных сведений об этом драйвере посетите веб-узел Kodak. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
