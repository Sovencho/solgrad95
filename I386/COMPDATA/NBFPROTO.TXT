Сведения о совместимости

Установленный драйвер протокола NETBEUI несовместим с Microsoft Windows ХР и будет удален при обновлении. Данный протокол исключен из новой версии Windows.

Для получения дополнительных сведений о данном драйвере посетите веб-узел производителя по адресу http://www.microsoft.com. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень протоколов, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
