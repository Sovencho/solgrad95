Сведения о совместимости

Программа Yamaha SoftSynthesizer S-YXG50 несовместима с данной версией Microsoft Windows и будет отключена при обновлении. Настоятельно рекомендуется удалить данное программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений о данных программах или версий, работающих в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
