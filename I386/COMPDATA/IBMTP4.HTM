Установленный драйвер устройства TrackPoint несовместим с Microsoft Windows ХР и будет отключен при обновлении. Для полного устранения конфликта данное устройство следует удалить.

Чтобы удалить это устройство, выполните следующие действия

  1. Нажмите кнопку Пуск и в меню Настройка выберите команду Панель управления.
  2. Дважды щелкните значок Установка и удаление программ.
  3. Нажмите кнопку Заменить/удалить и выберите программу IBM TrackPoint Support.
  4. Нажмите кнопку Заменить/Удалить, а затем OK.
  5. Перезагрузите компьютер и запустите программу установки Windows ХР.

Для получения дополнительных сведений об этом драйвере посетите веб-узел разработчика http://go.microsoft.com/fwlink/?linkid=386. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
