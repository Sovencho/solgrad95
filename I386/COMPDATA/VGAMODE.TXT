Сведения о совместимости

Данное устройство отображения в Microsoft Windows XP не поддерживается. Если это устройство совместимо со стандартом VESA, данный компьютер может использоваться для большинства задач, таких как работа в Интернете или чтение электронной почты. Некоторые задачи, такие как просмотр видеофильмов или компьютерные игры, возможно, не могут быть выполнены без установки драйвера экрана, совместимого с Microsoft Windows XP.

В некоторых случаях обновленный драйвер устройства может находиться на веб-узле Microsoft Windows Update.

Для получения дополнительных сведений о данном устройстве или драйвера, совместимого с Windows ХР, посетите веб-узел изготовителя.

Полный список устройств, поддерживаемых Windows ХР, содержится в списке совместимых устройств Microsoft Windows XP Compatibility List.
