Сведения о совместимости

Установленная версия программного обеспечения Microsoft Intellipoint несовместима с Microsoft Windows ХР и будет отключена при обновлении. Если установлена версия 3.1 или более поздняя, необходимо переустановить это программное обеспечение после обновления для правильной работы в среде Windows ХР.

Несмотря на то, что данное программное обеспечение не влияет на процесс обновления, рекомендуется удалить эту программу перед продолжением.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде Windows ХР, посетите веб-узел корпорации Майкрософт http://www.microsoft.com/mouse.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
