Сведения о совместимости

Программное обеспечение TBS Daytona несовместимо с Microsoft Windows ХР и будет отключено при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Для получения дополнительных сведений о данных программах или версии, работающей в среде Microsoft Windows ХР, посетите веб-узел изготовителя Voyetra Turtle Beach, Inc. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
