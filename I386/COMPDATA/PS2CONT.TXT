Сведения о совместимости

Установленный драйвер Contour Mouse несовместим с Microsoft Windows ХР и будет отключен при обновлении. Следует удалить драйверы и связанное с ними программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений об этом драйвере посетите веб-узел разработчика http://go.microsoft.com/fwlink/?LinkId=565. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
