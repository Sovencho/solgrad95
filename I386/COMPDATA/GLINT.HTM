Установленный драйвер видеоадаптера на базе Glint несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Примечание.

  • Для видеоадаптера на основе набора микросхем Permedia 2 программа установки Windows ХР добавит драйвер, поставляемый корпорацией Microsoft.
  • Для видеоадаптеров на основе Glint после завершения установки Windows XP монитор будет работать в режиме VGA.

Для получения дополнительных сведений о данном драйвере обратитесь к разработчику.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
