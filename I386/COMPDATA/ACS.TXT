Сведения о совместимости

Windows XP Server больше не поддерживает дополнительный компонент системы обеспечения качества обслуживания (QoS) службу контроля допуска (ACS). Программа установки не может выполнить обновление, если задействована служба контроля допуска (ACS). Удалите систему ACS и снова запустите программу установки. Для удаления ACS необходимы привилегии администратора.

Удаление ACS не повлияет на функционирование системы QoS и установку систем управления политикой других поставщиков. RSVP (Resource Reservation Setup Protocol, протокол настройки резервирования ресурсов) поддерживает другие системы управления политикой, совместимые с имеющимися опубликованными стандартами.

Чтобы удалить службу контроля допуска ACS:

1. На панели управления дважды щелкните значок «Сетевые подключения».
2. В меню «Дополнительно» выберите команду «Дополнительные сетевые компоненты».
3. Дважды щелкните значок «Сетевые службы».
4. Снимите флажок службы контроля доступа QoS и нажмите кнопку «OK».
5. Для удаления ACS нажмите кнопку «Далее», а затем — кнопку «Готово».
