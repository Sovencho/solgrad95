Сведения о совместимости

Программное обеспечение IBM PC System Monitor несовместимо с данной версией Microsoft Windows и будет отключено при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.
