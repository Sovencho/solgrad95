Сведения о совместимости

Возможно, для обеспечения правильной работы в среде Microsoft Windows ХР потребуется обновление установленной версии программного обеспечения Microsoft Systems Management Server (SMS).

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Windows, посетите страницу Microsoft Systems Management Server на веб-узле корпорации Майкрософт. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
