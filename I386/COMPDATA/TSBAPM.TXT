Сведения о совместимости

Установленная программа управления питанием Toshiba Power Management System несовместима с данной версией Microsoft Windows и будет отключена при обновлении. Большинство аналогичных функций управления электропитанием содержится непосредственно в Windows XP. Рекомендуется удалить данное программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
