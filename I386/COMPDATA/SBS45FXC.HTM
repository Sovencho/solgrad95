Существующее приложение Microsoft для работы с факсом не будет работать после обновления до Windows XP. Удалите приложение с помощью значка «Установка и удаление программ» на панели управления, а затем повторно запустите программу установки.

Для использования данного приложения для работы с факсом после обновления до Windows XP необходимо загрузить обновленную версию приложения. Обновленная версия доступна на веб-узле http://www.microsoft.com. Для данной версии приложения необходимо изменить настройки для работы с факсом.

Полный список программ, поддерживаемых в данной версии Microsoft Windows, содержится в списке совместимых программ на веб-узле корпорации Майкрософт.
