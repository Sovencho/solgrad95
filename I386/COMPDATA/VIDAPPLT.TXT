Сведения о совместимости

Программное обеспечение установленного видеоадаптера несовместимо с данной версией Microsoft Windows и будет отключено при обновлении.

Для получения дополнительных сведений о данном программном обеспечении обратитесь к изготовителю устройства или посетите его веб-узел.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
