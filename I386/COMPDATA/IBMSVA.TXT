Сведения о совместимости

Программа IBM SystemView Agent DMI Service Layer на данном компьютере может быть несовместима с этой версией Microsoft Windows и будет отключена при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Для получения дополнительных сведений о данном программном обеспечении посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
