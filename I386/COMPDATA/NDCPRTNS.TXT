Сведения о совместимости

Установленный драйвер для беспроводных устройств 802.11 несовместим с данной версией Microsoft Windows и будет отключен при обновлении.

Текущая конфигурация будет сохранена. Средство изменения конфигурации после обновления работать не будет. Конфигурацию можно будет изменить на вкладке «Дополнительно» окна свойств диспетчера устройств.

Корпорации Майкрософт не удается определить изготовителя этого устройства. Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List по адресу http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
