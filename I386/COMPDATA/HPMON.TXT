Сведения о совместимости

Монитор порта Hewlett Packard Network Port недоступен в данной версии Microsoft Windows. При использовании данного монитора порта определите, поддерживает ли устройство печать TCP/IP и измените параметры конфигурации порта для каждой очереди печати после завершения установки. Если перечень протоколов, поддерживаемых данным устройством, неизвестен, посетите веб-узел изготовителя для получения дополнительных предложений, совместимых с монитором порта.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
