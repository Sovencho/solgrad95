Сведения о совместимости

Установленный драйвер служебной программы Phoenix Power Management несовместим с данной версией Microsoft Windows и будет отключен при обновлении. Следует удалить эту программу для полного устранения конфликта.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
