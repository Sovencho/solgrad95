На данном компьютере выполняется служба Microsoft Site Server Internet Locator Service (ILS). Данная служба недоступна в среде Microsoft Windows XP и будет отключена при обновлении. Следовательно, если используются приложения, для которых необходима служба ILS, следует обеспечить наличии в организации хотя бы одного сервера, на котором может выполняться данная служба. Например, использование сервера ILS может быть необходимым в следующих случаях:

  • Используется программа NetMeeting версии 2.11 или более ранней.
  • Используются приложения на основе TAPI 3.0, такие как Phone Dialer.

Windows XP includes TAPI version 3.1. В TAPI 3.1 поддерживается ILS и разделы каталога приложений TAPI. Если планируется установить Windows ХР на все серверы и, следовательно, лишиться возможности запуска службы ILS, необходимо настроить раздел каталога приложения на контроллер домена Windows XP для обеспечения работы. Для получения дополнительных сведений обратитесь к статье базы знаний Q280319, «Error Message Upgrading Computer That Is Running the Microsoft Site Server Internet Locator Service (ILS)».
