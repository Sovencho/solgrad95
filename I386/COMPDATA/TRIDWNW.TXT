Сведения о совместимости

Установленный драйвер звуковой платы Trident 4D Wave несовместим с Microsoft Windows ХР и будет отключен при обновлении.

Для получения дополнительных сведений об этом драйвере посетите веб узел изготовителя Trident Microsystems Inc. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
