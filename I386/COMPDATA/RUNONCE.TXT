Сведения о совместимости

Microsoft Windows ХР обнаружила на компьютере не полностью установленную программу. Установка Windows ХР не может быть продолжена до тех пор, пока не завершена установка этой программы.

Чтобы завершить процесс установки этой программы

1. Закройте программу установки Windows ХР.
2. Выйдете из системы и войдите снова.
3. После входа в систему программа запустится немедленно, позволяя завершить установку.
4. Перезапустите программу установки Windows ХР.
