Сведения о совместимости

Некоторые версии программы Olympus CAMEDIA SmartDisk Flash Path несовместимы с Windows. Программа установки не может определить, совместима ли установленная версия программного обеспечения CAMEDIA с новой версией Windows.

При установке Windows программное обеспечение Olympus CAMEDIA будет отключено. После установки Windows установите версию программы, совместимую с новой версией Windows.

Для получения дополнительных сведений о данном драйвере обратитесь к изготовителю устройства или посетите его веб-узел.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
