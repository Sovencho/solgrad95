Сведения о совместимости

В процессе обновления установленное программное обеспечение Award Cardware будет заменено обновленной версией, совместимой с данной версией Microsoft Windows. При обновлении предыдущая версия будет удалена.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя.

Полный перечень программ, поддерживаемых данной версией Windows, содержится в списке Microsoft Windows XP Product Compatibility List на веб-узле корпорации Майкрософт.
