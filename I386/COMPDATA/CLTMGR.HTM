Программное обеспечение Toshiba Client Manager, установленное на этом компьютере, несовместимо с данной версией Windows. При обновлении он будет удален и заменен совместимой версией. Перед продолжением обновления рекомендуется удалить программу Toshiba Client Manager с помощью значка «Установка и удаление программ» на панели управления.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
