Сведения о совместимости

Программа установки Microsoft Windows ХР обнаружила дисковод Iomega Zip, подключенный к параллельному порту компьютера. Если дисковод Iomega Zip подключен к параллельному порту компьютера, после обновления некоторые установленные средства работы с дисководом Iomega Zip не будут работать должным образом.

Несмотря на то, что Windows ХР содержит встроенную поддержку дисковода Iomega Zip, драйверы этого устройства будут отключены при обновлении. Рекомендуется удалить средства работы с дисководом Iomega Zip перед продолжением.

Для получения дополнительных сведений посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
