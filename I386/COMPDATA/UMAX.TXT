Сведения о совместимости

Установленный драйвер сканера UMAX VistaScan версии 2.40 несовместим с данной версией Microsoft Windows и будет отключен при обновлении. Программа установки Windows ХР установит новый драйвер UMAX DS для правильной работы сканера после завершения обновления. Возможно, компания Umax разработала более новую версию программы VistaScan, работающую в среде Windows ХР.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
