Программа установки обнаружила на компьютере средства администрирования Windows 2000. Перед обновлением следует удалить средства администрирования Windows 2000, воспользовавшись компонентом панели управления «Установка и удаление программ». Для удаленного администрирования служб сервера с компьютера, работающего под управлением Windows XP Professional, используется средство «Дистанционное управление рабочим столом».

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
