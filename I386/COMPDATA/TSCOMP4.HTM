Программа установки Microsoft Windows XP обнаружила, что на данном компьютере выполняется сервер терминалов (формально — «Службы терминалов в режиме сервера приложений»). Сервер терминалов в среде Windows XP Server не поддерживается. Чтобы выполнить обновление на данном компьютере и продолжить использование сервера терминалов, необходимо отменить данное обновление и установить Windows XP Advanced Server. Кроме того, сервер терминалов входит в состав Windows XP Datacenter Server.
