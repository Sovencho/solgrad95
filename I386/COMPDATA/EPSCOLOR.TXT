Сведения о совместимости

Установленные в данный момент драйверы следующих принтеров Epson, возможно, подключенных к данному компьютеру, под управлением Microsoft Windows XP будут функционировать неправильно:

EPSON Stylus COLOR 1520
EPSON Stylus COLOR 400
EPSON Stylus COLOR 600
EPSON Stylus COLOR 850
EPSON Stylus COLOR 850ne
EPSON Stylus COLOR 860
EPSON Stylus Scan 2500
EPSON Stylus COLOR 300
EPSON Stylus COLOR 800
EPSON Stylus COLOR 3000

Если данный драйвер будет уделен перед обновлением, программа установки Windows XP установит совместимый драйвер.
Для получения последней версии драйверов принтера посетите веб-узел Epson. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.
