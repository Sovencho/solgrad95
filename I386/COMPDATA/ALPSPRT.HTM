Установленные в данный момент драйверы следующих принтеров Alps, возможно, подключенных к данному компьютеру, под управлением Microsoft Windows XP будут функционировать неправильно:

  • Alps MD-1000
  • Alps MD-1300
  • Alps MD-2010
  • Alps MD-2300
  • Alps MD-4000
  • Alps MD-5000

Замените драйвер обновленной версией, в противном случае программа установки Windows XP установит совместимый драйвер.

Для получения последней версии драйвера посетите веб-узел Alps. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.
