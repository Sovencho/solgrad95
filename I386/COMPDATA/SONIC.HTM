Установленный драйвер звуковой платы S3 Sonic Vibes несовместим с Microsoft Windows ХР и будет отключен при обновлении.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя S3 Incorporated. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
