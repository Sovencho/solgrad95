Сведения о совместимости

Установленные драйверы камер Creative Video Blaster WebCam Go и Go Plus несовместимы с программой установки Microsoft Windows XP. При обновлении камера ДОЛЖНА БЫТЬ «ОТКЛЮЧЕНА».

Для получения дополнительных сведений об этих драйверах посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
