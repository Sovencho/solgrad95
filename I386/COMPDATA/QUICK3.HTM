Программное обеспечение Quicktime 3 несовместимо с данной версией Windows. Рекомендуется обновить эту программу до Quicktime 4 перед установкой Windows ХР.

Для получения дополнительных сведений о данной программе или обновления до Quicktime 4 посетите веб-узел разработчика.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
