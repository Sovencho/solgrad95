Сведения о совместимости

Программа Handwriter версии 2.0 компании CIC несовместима с Microsoft Windows ХР.

Приложение не будет работать, а при запуске системы на экран будет выводиться сообщение об ошибке следующего содержания:

CIC	Не удается установить приоритет

Эта ошибка не препятствует установке Windows ХР, а программу CIC Handwriter можно удалить после завершения установки Windows ХР.

Для получения дополнительных сведений о данном устройстве или версии программы, работающей в среде Windows XP, посетите веб-узел разработчика http://go.microsoft.com/fwlink/?linkid=387. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
