Программа установки Microsoft Windows ХР не может определить версию установленного на компьютере программного обеспечения Microsoft Windows CE Services.

Программное обеспечение Microsoft Windows CE Services 2.2 необходимо для поддержания связи с Windows XP. Если на компьютере установлена более ранняя версия Windows CE, использование приложений Windows CE будет невозможно до тех пор, пока не выполнено обновление до версии 2.2. Обновить Windows CE можно до или после установки Windows ХР.

Для получения дополнительных сведений посетите страницу Microsoft Windows CE Services 2.2 на веб-узле корпорации Майкрософт
