Установка пакета обновления 3 (SP3) для Microsoft Windows XP отменена, так как невозможно установить пакет обновления 3 (SP3) для Windows XP поверх пакета многоязыкового интерфейса пользователя для протокола удаленного рабочего стола 6.0 (обновление для Windows XP KB925877). Чтобы продолжить, удалите обновление KB925877.
