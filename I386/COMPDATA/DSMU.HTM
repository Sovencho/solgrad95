Программное обеспечение Directory Service Migration Utility (DSMU) несовместимо с данной версией Microsoft Windows. Рекомендуется удалить данное программное обеспечение перед продолжением обновления. Нельзя удалить служебную программу Gateway Service для NetWare (GSNW) или IPX, предварительно не удалив DSMU.

Чтобы удалить DSMU:

  1. Нажмите кнопку Пуск, выберите пункт Настройка, выберите пункт Панель управления, а затем дважды щелкните значок Установка и удаление программ.
  2. Нажмите кнопку Добавление и удаление компонентов Windows.
  3. Выберите Dsmigrate для удаления.
  4. Вручную удалите все файлы из каталога \program files\dsmigrat.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
