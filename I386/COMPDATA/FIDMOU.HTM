Для использования всех функций программное обеспечение для сенсорной панели Fujitsu Takamisawa должно быть повторно установлено после обновления.

Для получения дополнительных сведений об этом программном обеспечении посетите веб-узел изготовителя. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный список устройств, поддерживаемых Windows ХР, содержится в списке совместимых устройств Microsoft Windows XP Compatibility List.
