Программное обеспечение Microsoft SNA Server несовместимо с данной версией Microsoft Windows. Для использования данного программного обеспечения в среде Windows XP необходимо установить пакет обновления SNA Server 4.0 Service Pack версии 3 или более поздней.

Для получения дополнительных сведений о данной программе и загрузки пакета обновления SNA Server 4.0 Server Pack 3 посетите страницу Microsoft SNA Server на веб-узле корпорации Майкрософт. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
