Установленный драйвер Canon Canoscan 620P должен быть повторно установлен после обновления Microsoft Windows.

Для получения дополнительных сведений об этом драйвере посетите веб-узел Canon. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
