Установленные в данный момент драйверы следующих принтеров Compaq, возможно, подключенных к данному компьютеру, под управлением Microsoft Windows XP будут функционировать неправильно:

  • Compaq IJ700
  • Compaq IJ900

Замените драйвер обновленной версией, в противном случае программа установки Windows XP установит совместимый драйвер.

Для получения последней версии драйверов принтера посетите веб-узел Compaq. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.
