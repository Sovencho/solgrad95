Программное обеспечение устройства Ensoniq AudioPCI Mixer несовместимо с Microsoft Windows ХР и будет отключено при обновлении. Рекомендуется удалить данное программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений о данной программе или верси, работающей в среде данной версии Microsoft Windows, посетите веб-узел Creative Technology Ltd. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого Windows XP, содержится в списке Microsoft Windows XP Product Compatibility List на веб-узле корпорации Майкрософт
