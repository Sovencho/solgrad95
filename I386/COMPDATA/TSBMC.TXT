Сведения о совместимости

Программное обеспечение Toshiba Mouse Control несовместимо с Microsoft Windows ХР и будет отключено при обновлении. Рекомендуется удалить данное программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений о данном программном обеспечении или версии программы, работающей в среде Windows XP, посетите веб-узел разработчика http://go.microsoft.com/fwlink/?linkid=385. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
