Сведения о совместимости

Установленный драйвер устройства IBM 9397 Video Capture несовместим с данной версией Microsoft Windows и будет отключен при обновлении. Для полного устранения конфликта данное устройство следует удалить.

Чтобы удалить это устройство, выполните следующие действия

1. Нажмите кнопку «Пуск» и в меню «Настройка» выберите команду «Панель управления».
2. Дважды щелкните значок «Звуки и аудиоустройства».
3. На вкладке «Оборудование» дважды щелкните значок «Устройства записи видео».
4. Выберите устройство «IBM 9397 Video Capture».
5. Нажмите кнопку «Удалить», а затем — кнопку «Да».
6. Нажмите кнопку «OK»и перезагрузите компьютер.
7. Перезапустите программу установки Windows ХР.

Для получения дополнительных сведений об этом драйвере посетите веб-узел изготовителя.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
