При использовании установленного драйвера в Microsoft Windows ХР принтеры, использующие файл sdnt4.dll, будут работать неправильно. Имеется возможность заменить драйвер обновленной версией, в противном случае программа установки Windows XP установит совместимый драйвер.

Для получения последней версии драйвера посетите веб-узел изготовителя.
