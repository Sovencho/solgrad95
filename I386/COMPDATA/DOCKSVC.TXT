Сведения о совместимости

Программное обеспечение Ultra 2000 Docking Services несовместимо с данной версией Microsoft Windows и будет отключено при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программ, поддерживаемых данной версией Windows, содержится в списке Microsoft Windows XP Product Compatibility List на веб-узле корпорации Майкрософт.
