Сведения о совместимости

Программное обеспечение XLINK Omni NFS Gateway 4.0 несовместимо с данной версией Microsoft Windows. Следует удалить это программное обеспечение перед продолжением обновления.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
