Установленный драйвер для транспортного протокола DLC несовместим с данной версией Microsoft Windows и будет отключен при обновлении. Данный протокол исключен из новой версии Windows.

Драйвер DLC для Microsoft Windows XP доступен в продукте Host Integration Server 2000. Для получения дополнительных сведений посетите веб-узел http://www.microsoft.com/hiserver. Веб-адреса могут изменяться, поэтому существует вероятность того, что к этому узлу подключиться не удастся.

Полный перечень протоколов, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
