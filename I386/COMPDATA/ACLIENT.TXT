Сведения о совместимости

Программное обеспечение Altiris Client Service несовместимо с Microsoft Windows XP и будет отключено при обновлении. Следует удалить это программное обеспечение для полного устранения конфликта.

Для получения дополнительных сведений о данных программах или версий, работающих в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого Windows XP, содержится в списке Microsoft Windows XP Product Compatibility List на веб-узле корпорации Майкрософт
