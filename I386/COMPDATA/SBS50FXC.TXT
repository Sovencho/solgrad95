Сведения о совместимости

Существующее приложение Microsoft для работы с факсом не будет работать после обновления до Windows XP. Удалите приложение для работы с факсом с помощью значка «Установка и удаление программ» на панели управления, а затем повторно запустите программу установки.

После установки Windows XP добавьте соединение с факсом для повторного подключения к факс серверу и используйте приложение Windows XP Fax для отправки и получения факсов через данное соединение.

Полный список программ, поддерживаемых в данной версии Microsoft Windows, содержится в списке совместимых программ на веб-узле корпорации Майкрософт.
