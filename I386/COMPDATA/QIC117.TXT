Сведения о совместимости

Программа установки Microsoft Windows ХР обнаружила на этом компьютере накопитель на магнитной ленте, подключенный к контроллеру гибких дисков. Это устройство несовместимо с Microsoft Windows ХР и будет недоступно после обновления.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
