ПРЕДУПРЕЖДЕНИЕ!

Прочтите этот текст перед продолжением обновления!

Программа установки Windows ХР обнаружила в конфигурации системы RAID-контроллер Zero-Channel (SCSI Interrupt Steering Logic). Имеется возможность продолжить обновление, однако установка может быть не завершена, а данные конфигурации могут быть потеряны.

Обратитесь к изготовителю компьютера для получения сведений об обновлении до Windows XP.

Полный перечень устройств, поддерживаемых данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
