Сведения о совместимости

Программа Toshiba Automatic Shutdown Subsystem несовместима с данной версией Microsoft Windows и должна быть удалена перед продолжением обновления.

Чтобы удалить программу Toshiba Automatic Shutdown Subsystem, выполните следующие действия.

1. Нажмите кнопку «Пуск» и выберите команды «Программы», «Стандартные» и «Командная строка».
2. Выполните одно из следующих действий.

При использовании серии Toshiba «GS» введите > SETUPASD REMOVE.
При использовании серии Toshiba «MAGNIA» введите > SETUPASD UNINSTALL [-NOISC]. Параметр [-NOISC] следует вводить при использовании Intel LDSM и ISC.

Для получения дополнительных сведений о данном программном обеспечении или версии, работающей в среде данной версии Microsoft Windows, посетите веб-узел изготовителя.

Полный перечень программного обеспечения, поддерживаемого данной версией Microsoft Windows, содержится в списке Microsoft Windows XP Compatibility List на веб-узле http://www.microsoft.com.
